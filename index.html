General

Основные системные функции инициализации и игровых циклов. Core system initialization and core loop functions.

is_game_ready()
Возвращает true, если игра инициализирована и готова. Returns true if the game is initialized and ready.
on_render()
(Callback) Определяйте эту функцию для выполнения кода каждый кадр. (Callback) Define this function to run code every frame.

Drawing

Инструменты рендеринга геометрии и текста на экране. Screen overlay, rendering geometry and text drawing tools.

draw_text(x, y, r, g, b, a, text)
Рисует текст на экране по заданным координатам и цвету. Draws text on the screen using 2D coordinates and RGBA color.
draw_line(x1, y1, x2, y2, r, g, b, a, thickness)
Отрисовывает линию с указанной толщиной. Draws a line with custom thickness.
draw_circle(x, y, radius, r, g, b, a, segments)
Рисует круг с разбиением на сегменты. Draws a circle. Segments parameter defines complexity/smoothing.
draw_circle_3d(x, y, z, radius, r, g, b, a, segments)
Рисует 3D круг в позиции мира. Draws a 3D circle at world position.
draw_rect(x, y, w, h, r, g, b, a, thickness)
Рисует контур прямоугольника. Draws a rectangle outline.
draw_filled_rect(x, y, w, h, r, g, b, a)
Рисует залитый прямоугольник. Draws a filled rectangle.
world_to_screen(x, y, z)
Переводит 3D координаты мира в 2D координаты экрана. Возвращает {x, y} или null. Projects 3D world coordinates onto 2D screen coordinates. Returns {x, y} or null.
get_screen_center()
Возвращает 2D координаты центра экрана {x, y}. Returns the 2D coordinates of the screen center {x, y}.
get_camera_position()
Возвращает 3D координаты камеры {x, y, z}. Returns the 3D coordinates of the camera {x, y, z}.

Game Objects

Взаимодействие со структурами сущностей и игроков. Methods to interact with actors, entity structures and players.

get_objects()
Возвращает массив указателей на все объекты (акторы) в мире. Returns an array of pointers to all objects (actors) in the world.
get_object_name(obj)
Возвращает имя объекта или актора по его указателю. Returns the name of the object or actor by its pointer.
get_local_player()
Возвращает указатель на локального игрока или null. Returns the pointer to the local player pawn, or null.
get_my_controller()
Возвращает указатель на локальный контроллер игрока или null. Returns the pointer to the local player controller, or null.
get_targets()
Возвращает массив указателей на все валидные цели (игроков/акторов). Returns an array of pointers to all valid targets (players/actors).
get_name(ptr)
Возвращает имя сущности по ее указателю. Returns the name of the actor by its pointer.
get_location(ptr)
Возвращает 3D позицию объекта {x, y, z}. Returns the 3D location {x, y, z} of the actor.
get_health(ptr)
Возвращает количество здоровья актора. Returns the health value of the actor.
is_friend(ptr)
Проверяет, находится ли актор в списке друзей (true/false). Returns true if the actor is on your friend list.

UI Management

Управление встроенным меню софта, создание вкладок и интерактивных элементов. Menu interaction, tab creations, sliders, comboboxes and layout bindings.

register_tab(name)
Создает новую главную вкладку в сайдбаре меню. Creates a new main tab in the menu sidebar.
register_section(tab_name, section_name)
Создает подсекцию внутри вкладки. Creates a sub-section inside a tab.
add_checkbox(tab_name, section_name, label, default_value)
Добавляет интерактивный чекбокс (boolean). Adds a custom checkbox setting element (boolean).
add_slider(tab_name, section_name, label, default, min, max)
Добавляет ползунок для выбора числовых значений с плавающей запятой (number). Adds a slider for numerical configuration (number).
add_combo(tab_name, section_name, label, default_index, items_array)
Добавляет выпадающий список вариантов. Возвращает индекс выбранного элемента (number). Adds a dropdown selection element. Returns selected element index (number).
add_colorpicker(tab_name, section_name, label, r, g, b, a)
Добавляет палитру выбора цвета. Возвращает объект вида {r, g, b, a}. Adds an interactive color picker element. Returns a color object {r, g, b, a}.
get_ui_value(label)
Возвращает текущее состояние элемента UI в зависимости от его типа. Returns the current value of a UI element depending on its type.

Advanced SDK API

Прямая работа с движком игры и вызовами функций. Low-level engine interactions and direct function calling.

find_class(name)
Находит указатель на класс по его названию. Finds a class pointer by name (e.g., "DamageRegular").
call_function(obj_ptr, func_name, args_array)
Вызывает любую SDK функцию через встроенный ProcessEvent. Calls any SDK function via ProcessEvent mechanism.
get_property(obj_ptr, prop_name)
Читает любое свойство/переменную из объекта. Reads any variable from an object by its SDK name.
set_property(obj_ptr, prop_name, value)
Записывает значение в любую переменную объекта. Writes a value to any object variable.

Native FFI

Динамический Foreign Function Interface (FFI) для прямого взаимодействия с Windows API подсистемами. Dynamic Foreign Function Interface (FFI) to interact directly with low-level Windows API subsystems.

native.define(name, dll, function, returnType, ...argTypes)
Регистрирует внешнюю C-функцию из указанной DLL библиотеки для последующего вызова. Registers an exported C-function from a specified DLL library for subsequent execution.
native.call(name, ...args)
Вызывает ранее определенную Windows API функцию по её зарегистрированному имени. Invokes a previously defined Windows API function by its registered configuration name.
Supported Types / Поддерживаемые типы данных
При связывании аргументов и возвращаемых значений используйте строго следующие строковые типы: When mapping arguments and return types, strictly use the following syntax values:
  • "void" - отсутствие возвращаемого значенияno return value
  • "int" - целочисленное значение типа intstandard integer values
  • "ptr" / "pointer" - указатели на области памяти или структурыmemory addresses and pointers
  • "string" / "str" - строковые значения (C-style char pointer)string sequences
  • "float" - число с плавающей точкой (одинарная точность)floating point numbers
  • "double" - число с плавающей точкой (двойная точность)double precision floats

Native Windows API FFI Usage

native.define("MessageBoxA", "user32.dll", "MessageBoxA", "int", "ptr", "string", "string", "int");
native.define("Beep", "kernel32.dll", "Beep", "int", "int", "int");
native.define("Sleep", "kernel32.dll", "Sleep", "void", "int");
native.define("GetTickCount", "kernel32.dll", "GetTickCount", "int");

function showMessage(text, title) {
    return native.call("MessageBoxA", null, text, title, 0);
}

function playBeep(freq, duration) {
    native.call("Beep", freq, duration);
}

function sleep(ms) {
    native.call("Sleep", ms);
}

function getTickCount() {
    return native.call("GetTickCount");
}

showMessage("Hello from native FFI!", "Test");

playBeep(1000, 200);

function on_render() {
    var ticks = getTickCount();
    draw_text(10, 140, 255, 255, 0, 255, "System ticks: " + ticks);
}

Carcass Finder ESP

function on_render() {
    if (!is_game_ready()) return;
    var objects = get_objects();
    for (var i = 0; i < objects.length; i++) {
        var obj = objects[i];
        var name = get_object_name(obj);
        if (name.indexOf("Carcass") !== -1) {
            var loc = get_location(obj);
            if (loc) {
                var screen = world_to_screen(loc.x, loc.y, loc.z);
                if (screen) {
                    draw_text(screen.x, screen.y, 255, 255, 255, 255, name);
                }
            }
        }
    }
}

Local Player Features

function on_render() {
    if (!is_game_ready()) return;

    var me = get_local_player();
    if (!me) return;

    var loc = get_location(me);
    var hp = get_health(me);
    var name = get_name(me);

    if (loc) {
        var screen = world_to_screen(loc.x, loc.y, loc.z);
        if (screen) {
            draw_text(screen.x, screen.y, 0, 255, 0, 255, name + " (" + Math.floor(hp) + " HP)");
        }
    }
}

Render UI & Menu Bindings

register_tab("My Custom ESP");
register_section("My Custom ESP", "Settings");

add_checkbox("My Custom ESP", "Settings", "Enable ESP", true);
add_slider("My Custom ESP", "Settings", "Max Distance", 150.0, 10.0, 500.0);
add_combo("My Custom ESP", "Settings", "ESP Type", 0, ["2D Box", "3D Circle", "Text Only"]);
add_colorpicker("My Custom ESP", "Settings", "Visual Color", 255, 255, 0, 255);

function on_render() {
    if (!is_game_ready()) return;

    var isEnabled = get_ui_value("Enable ESP");
    var maxDist = get_ui_value("Max Distance");
    var espTypeIndex = get_ui_value("ESP Type");
    var clr = get_ui_value("Visual Color");

    if (!isEnabled) return;

    var targets = get_targets();
    var me = get_local_player();
    if (!me) return;
    
    var myLoc = get_location(me);
    if (!myLoc) continue;

    for (var i = 0; i < targets.length; i++) {
        var actor = targets[i];
        if (actor === me) continue;

        var loc = get_location(actor);
        if (!loc) continue;

        var dx = loc.x - myLoc.x;
        var dy = loc.y - myLoc.y;
        var dz = loc.z - myLoc.z;
        var distMeters = Math.sqrt(dx*dx + dy*dy + dz*dz) / 100;

        if (distMeters > maxDist) continue;

        var screen = world_to_screen(loc.x, loc.y, loc.z);
        if (!screen) continue;

        var name = get_name(actor);

        if (espTypeIndex === 1) {
            draw_circle_3d(loc.x, loc.y, loc.z, 50.0, clr.r, clr.g, clr.b, clr.a, 16);
        }
        
        draw_text(screen.x, screen.y, clr.r, clr.g, clr.b, clr.a, name + " [" + Math.floor(distMeters) + "m]");
    }
}

3D Circles ESP & Snaplines

register_tab("night");
register_section("night", "esp_3d");
add_checkbox("night", "esp_3d", "Enable 3D Visuals", false);

function on_render() {
    if (!is_game_ready()) return;

    if (get_ui_value("Enable 3D Visuals")) {
        var targets = get_targets();
        var center = get_screen_center();

        for (var i = 0; i < targets.length; i++) {
            var player = targets[i];
            if (is_friend(player)) continue;

            var loc = get_location(player);
            if (!loc) continue;

            draw_circle_3d(loc.x, loc.y, loc.z, 60.0, 0, 255, 0, 255, 32);

            var screen = world_to_screen(loc.x, loc.y, loc.z);
            if (screen && center) {
                draw_line(center.x, center.y, screen.x, screen.y, 0, 255, 0, 150, 1.0);
            }
        }
    }
}
